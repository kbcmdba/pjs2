# PHP Job Seeker 2

PJS2 is a job search tracking tool that helps candidates track their searches,
companies, specific jobs, contacts, and application statuses.

## Requirements

- Apache or a similar web server with PHP module
- PHP 5.6 or later
- MySQL 5.6 or later
